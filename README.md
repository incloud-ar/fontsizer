FontSizer
=========

Jquery plugin for increase/decrease font size

Usage
=========
```
$(function() {
  $('#controls a').fontsizer();
});
```

// Options
```
$('#controls a').fontsizer({
  min: 12, // Min font size
  max: 18, // Max font size
  elements: 'p, div, span, .class', // elements to resize 
});
```
